# Pertussis in the United States: Early Vaccine Impact and Modern Resurgence

Exploratory data analysis of historical pertussis incidence in the United States, investigating disease trends, vaccination impact, and recent resurgence using epidemiological data and statistical visualization.

This project was completed as part of a two-person university assignment for the *Data Visualization* course in the Master's programme *Life Science Informatics* at Deggendorf Institute of Technology.

The repository primarily showcases my work in developing the R Markdown workflow, integrating the analysis into a reproducible report, and scientific data visualization.
